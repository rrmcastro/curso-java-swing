Lista de comandos (Terminal Bash)
pwd - apresenta o caminho da pasta
ls - lista o conteudo da pasta
mkdir - criar nova pasta
ctrl + l - limpa a tela do bash
rm -r - excluir para sem mensagem para o usuario



comandos git
git log --oneline - lista todos do commits de uma branch
git status - mostra as alterações feitas no projeto
git commit -m  - criar um novo commit com mensagem
