Comandos Git

Lista de comandos (Terminal Bash)
pwd - apresenta o caminho da pasta
ls - lista o conteudo da pasta
mkdir - criar nova pasta
ctrl + l - limpa a tela do bash
rm -r - excluir para sem mensagem para o usuario



comandos git
git log --oneline - lista todos do commits de uma branch
git status - mostra as alterações feitas no projeto
git add . - ira adicionar todos os arquivos alterado ou novos
git commit -m  - criar um novo commit com mensagem 
git push origin - envia os arquivos alterado para o servidor
git restore --staged - remover o arquivo da stage area
history - lista todos os ultimos comandos feitos
git checkout -b  <nome-branch> - criar uma nova branch
git branch -l - listar as branchs locais
git branch -D <nome-branch>
